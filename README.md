# ETS POLODIEU ELECTRONICS Frontend

Vite + React frontend ready for Vercel.

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

Create a `.env` file from `.env.example`.

```env
VITE_API_BASE_URL=https://your-backend-url.com/api
```

## Deploy to Vercel

1. Upload this project to GitHub.
2. Import the repo into Vercel.
3. Add environment variable:
   - `VITE_API_BASE_URL`
4. Deploy.

## What is included

- storefront homepage
- cart
- checkout
- MTN MoMo / Orange Money / Card / Cash on Delivery
- WhatsApp order button
- WhatsApp floating chat button
- admin login
- admin product CRUD
- image upload support via backend
- orders dashboard with status update buttons
- browser mode + online database mode
